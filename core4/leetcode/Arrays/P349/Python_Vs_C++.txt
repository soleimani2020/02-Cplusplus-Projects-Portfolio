| Feature / Aspect                      | C++ (`Solution_Sort`)                         | Python (`Solution`)                         | Notes / Difference                                                         |                                                                          |                                                                                       |
| ------------------------------------- | --------------------------------------------- | ------------------------------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------------------------------- |
| **Array / Vector**                    | `vector<int> nums1, nums2`                    | `list nums1, nums2`                         | C++ uses `std::vector`; Python uses `list`.                                |                                                                          |                                                                                       |
| **Sorting**                           | `sort(nums1.begin(), nums1.end());`           | `nums1.sort()`                              | C++ uses `std::sort` with iterators; Python uses built-in `sort()` method. |                                                                          |                                                                                       |
| **Result container**                  | `vector<int> result;`                         | `result = []`                               | Similar dynamic array behavior.                                            |                                                                          |                                                                                       |
| **Index initialization**              | `int i=0, j=0;`                               | `i, j = 0, 0`                               | Python allows tuple unpacking; C++ requires explicit type declaration.     |                                                                          |                                                                                       |
| **While loop**                        | `while(i < nums1.size() && j < nums2.size())` | `while i < len(nums1) and j < len(nums2):`  | `&&` in C++ vs `and` in Python; `size()` vs `len()`.                       |                                                                          |                                                                                       |
| **Increment indices**                 | `i++` / `j++`                                 | `i += 1` / `j += 1`                         | Python uses `+=`; C++ has `++` operator.                                   |                                                                          |                                                                                       |
| **Comparison**                        | `nums1[i] < nums2[j]`                         | Same                                        | Identical.                                                                 |                                                                          |                                                                                       |
| **Add to result avoiding duplicates** | `if(result.empty()                            |                                             | result.back() != nums1[i]) result.push_back(nums1[i]);`                    | `if len(result) == 0 or result[-1] != nums1[i]: result.append(nums1[i])` | `empty()` vs `len(result)==0`; `back()` vs `result[-1]`; `push_back()` vs `append()`. |
| **Return result**                     | `return result;`                              | `return result`                             | Semicolon required in C++.                                                 |                                                                          |                                                                                       |
| **Printing output**                   | `for (int x : res) cout << x << " ";`         | `for x in res: print(x, end=" ")`           | Python uses `end=" "` to avoid newlines.                                   |                                                                          |                                                                                       |
| **Main function**                     | `int main() { ... }`                          | No main function required; just script code | Python runs top-down; C++ requires `main()`.                               |                                                                          |                                                                                       |
| **Memory management**                 | Automatic for vectors; no GC, stack vs heap   | Fully automatic garbage collection          | Python handles dynamic memory automatically.                               |                                                                          |                                                                                       |
| **Time complexity**                   | `O(n log n + m log m)` (sorting)              | Same                                        | Algorithm complexity is identical.                                         |                                                                          |                                                                                       |
