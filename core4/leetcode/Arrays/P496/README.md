# LeetCode Problem 496: Next Greater Element I (Easy)

## 🧩 Problem Statement

You are given two integer arrays `nums1` and `nums2`:

- `nums1` is a subset of `nums2`.
- For each element in `nums1`, find the **next greater element** in `nums2`.
- The **next greater element** of an element `x` in `nums2` is the **first element to the right of `x` that is greater than `x`**.  
- If no such element exists, return `-1` for that number.
