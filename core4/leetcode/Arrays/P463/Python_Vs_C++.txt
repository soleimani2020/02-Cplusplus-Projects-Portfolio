| Aspect                                | Python                                                    | C++                                                              |
| ------------------------------------- | --------------------------------------------------------- | ---------------------------------------------------------------- |
| **Grid Declaration**                  | `grid = [[0,1,0],[1,1,1]]` → list of lists                | `vector<vector<int>> grid = {{0,1,0},{1,1,1}};`                  |
| **Loop over indices**                 | `for i in range(rows):`                                   | `for(int i=0; i<rows; i++)`                                      |
| **Accessing elements**                | `grid[i][j]`                                              | `grid[i][j]` (same)                                              |
| **If statement syntax**               | `if i>0 and grid[i-1][j]==1:` → colon `:` and indentation | `if(i>0 && grid[i-1][j]==1) perimeter--;` → braces `{}` optional |
| **Variable declaration**              | Dynamic typing → `perimeter = 4`                          | Static typing → `int perimeter = 4;`                             |
| **Subtraction / Increment**           | `perimeter -= 1`                                          | `perimeter--;`                                                   |
| **Adding to total**                   | `total += perimeter`                                      | `total += perimeter;`                                            |
| **Function definition**               | `def islandPerimeter(self, grid):`                        | `int islandPerimeter(vector<vector<int>>& grid)`                 |
| **Class instantiation & method call** | `sol = Solution(); sol.islandPerimeter(grid)`             | `Solution sol; sol.islandPerimeter(grid);`                       |
| **Print output**                      | `print("Island Perimeter:", perimeter)`                   | `cout << "Island Perimeter: " << perimeter << endl;`             |
| **Main function / entry point**       | `if __name__ == "__main__":`                              | `int main() { ... return 0; }`                                   |
| **Boundary check**                    | `i < rows - 1`                                            | `i < rows - 1` (same logic, same check)                          |
| **Memory management**                 | Automatic                                                 | Automatic for STL containers, manual for raw arrays              |
| **Semicolons**                        | ❌ Not needed                                              | ✅ Required at end of statements                                  |
| **Code length**                       | Shorter & cleaner                                         | Slightly longer & more verbose                                   |
| **Performance**                       | Slower (interpreted)                                      | Faster (compiled)                                                |
| **Error prone parts**                 | Typo in variable names crashes (like `premeter`)          | Compiler catches type errors & undeclared variables              |
