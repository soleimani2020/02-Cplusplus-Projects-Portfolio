# 697. Degree of an Array

## Problem Statement

Given a non-empty array of non-negative integers `nums`, the **degree** of this array is defined as the maximum frequency of any one of its elements.

Your task is to find the **smallest possible length of a contiguous subarray** of `nums` that has the same degree as the original array.

---

## Example 1

### Input

nums = [1,2,2,3,1]
