# LeetCode Problem 485: Max Consecutive Ones (Easy)

## 🧩 Problem Statement

Given a binary array `nums`, return the maximum number of consecutive `1`s in the array.

---

## 📌 Examples

### Example 1:

**Input:**
nums = [1,1,0,1,1,1]
**Output:**
3
