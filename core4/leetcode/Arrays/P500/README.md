# LeetCode Problem 500: Keyboard Row (Easy)

## 🧩 Problem Statement

Given a list of words, return the words that can be typed using letters of the alphabet **on only one row of a standard QWERTY keyboard**.

- Keyboard rows:
 
Row 1: "qwertyuiop"

Row 2: "asdfghjkl"

Row 3: "zxcvbnm"


---

## 📌 Examples

### Example 1:

**Input:**

words = ["Hello","Alaska","Dad","Peace"]

**Output:**

["Alaska","Dad"]
