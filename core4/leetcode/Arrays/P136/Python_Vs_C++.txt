| Feature / Aspect         | **Python Version**                      | **C++ Version**                                          | Notes / Key Differences                                                                                  |
| ------------------------ | --------------------------------------- | -------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| **Data Structure**       | `dict` (unordered key-value map)        | `unordered_map<int,int>`                                 | Both store number → frequency, but Python’s `dict` is built-in, C++ needs `#include <unordered_map>`     |
| **Initialization**       | `mp = {}`                               | `unordered_map<int,int> mp;`                             | Python is dynamically typed; C++ is strongly typed.                                                      |
| **Insert / Update**      | `mp[num] = mp.get(num,0) + 1`           | `mp[num]++`                                              | Python `.get()` avoids KeyError; in C++, `mp[num]` automatically inserts default `0` if key not present. |
| **Loop over numbers**    | `for num in nums:`                      | `for(int num : nums)`                                    | Python uses iterator syntax, C++ uses range-based for loop.                                              |
| **Condition**            | `if mp[num] == 1:`                      | `if(x == 1)` where `x = mp[nums[i]]`                     | Same logic; Python is more concise.                                                                      |
| **Return single number** | `return num`                            | `return nums[i]`                                         | Logic is identical.                                                                                      |
| **Empty input handling** | Not explicitly handled; can return `0`  | Not explicitly handled; can return `0`                   | You may add `if not nums: return 0` in Python, or `if(prices.empty()) return 0` in C++.                  |
| **Print / Output**       | `print(f"Single Number is : {result}")` | `cout << "single Number: " << object.singleNumber(nums)` | Python f-string vs C++ stream operator.                                                                  |
| **Space Complexity**     | O(n)                                    | O(n)                                                     | Both use a hashmap/dictionary.                                                                           |
|                          |                                         |                                                          |                                                                                                          |
