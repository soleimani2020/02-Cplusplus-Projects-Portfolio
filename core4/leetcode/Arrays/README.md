# Arrays Solutions for LeetCode Problems (C++)

This repository contains **object-oriented C++ solutions** for various **LeetCode array problems**.  
Each problem is organized in its own folder (`PXX`) with:

- A **README.md** describing the problem, examples, and solution approach.  
- A **main.cpp** file demonstrating the implementation using **object-oriented programming**.  

---

### Example: How to Add a New Problem

```bash
cd ~/02-Cplusplus-Projects-Portfolio/core4/leetcode/Arrays
mkdir P66
echo "# P66" > P66/README.md
touch P66/main.cpp
git add P66
git commit -m "Add P66 directory with README and main.cpp"
git pull origin main --rebase
git push origin main
