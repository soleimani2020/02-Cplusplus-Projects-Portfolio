# Unified Memory Debugger

This project merges three earlier memory tools into a single debugger:

- **01_Memory_Management_System** – raw allocator tracker
- **02_Memory_Track_System** – logical dataset tracker
- **03_SmartMemoryPool** – connection pool with smart pointers

## Features
- Real‑time memory allocation tracking
- Leak detection and reporting
- Object lifetime monitoring
- Connection pooling with automatic cleanup

## Build & Run
```bash
make
./unified_memory_debugger
